PROJECT SUMMARY

Automated CI/CD Deployment with Jenkins, Docker & K3s.

AUTOMATION:

  1. code push from git to github
  2. jenkins pull the code from github to store workspace
  3. then jenkins builds the docker image using dockerfile
  4. now docker image pushing into dockerhub
  5. finally k3s-server pull the image & run a container as pods
  6. check a webpage using k3s-server:nodeport in browser
